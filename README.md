# TodoApp (Ülesanded)

Простое приложение для задач на .NET MAUI.  
Можно добавлять, удалять и фильтровать задачи.  
Есть выбор темы (светлая / тёмная) и языка.

---

## Что можно делать
- Добавлять новую задачу  
- Указывать дату, время, приоритет и категорию  
- Удалять одну задачу или все сразу  
- Фильтровать по категории и приоритету  
- Менять тему приложения  
- Сохраняет все данные в базу (SQLite)

---

## Структура проекта

### **Models/**
- `TaskItem.cs` — одна задача (что нужно сделать)  
- `Priory.cs` — приоритеты: низкий, нормальный, высокий  

### **ViewModels/**
- `BaseViewModel.cs` — обновляет экран при изменениях  
- `TasksViewModel.cs` — логика задач (добавить, удалить, фильтр)  
- `SettingsViewModel.cs` — логика настроек  

### **Services/**
- `DatabaseService.cs` — база данных (SQLite)  
- `ThemeService.cs` — темы (светлая / тёмная)  
- `LocalizationService.cs` — язык приложения  

### **Views/**
- `TasksPage.cs` — экран со списком задач  
- `SettingsPage.cs` — экран настроек  

### **App/**
- `App.xaml` — стили и цвета  
- `App.xaml.cs` — запуск приложения  
- `AppShell.xaml` — вкладки (Ülesanded / Seaded)  
- `MauiProgram.cs` — подключение всех сервисов  

---

## Как работает
1. При запуске включается сохранённая тема и язык  
2. Загружается список задач  
3. Можно добавлять или удалять задачи  
4. Всё сохраняется в базу данных  
5. Настройки сохраняются при выходе  

---

## Технологии
- .NET MAUI  
- SQLite  
- MVVM (Model–View–ViewModel)
